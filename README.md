## Dataset Source

This project uses the PJM Hourly Energy Consumption dataset.

Source:
https://www.kaggle.com/datasets/robikscube/hourly-energy-consumption

Original data provided by PJM Interconnection LLC:
https://www.pjm.com/

## Dataset

The dataset contains hourly electricity consumption data from PJM Interconnection LLC, measured in megawatts (MW).

Source:
- PJM Interconnection
- Kaggle dataset by Rob Mulla

## Dataset Limitations
The regions have changed over the years so data may only appear for certain dates per region.
